## Project Overview

Static website for the **Treasure Valley Vintage Computing Club (TVVCC)** built with Hugo and deployed on AWS (S3 + CloudFront). The site uses a custom `1995` theme that outputs HTML 3.2 for compatibility with vintage browsers (Mosaic, Netscape Navigator, Lynx). CloudFront serves both HTTP and HTTPS (`allow-all` viewer protocol policy).

## Commands

### Local Development
```bash
hugo server
```

### Production Build
```bash
hugo --gc --minify --baseURL "https://tvvcc.org/"
```

### Create New Post
```bash
hugo new posts/my-post-title.md
```

## Architecture

- **hugo.toml** — Primary site configuration (base URL, theme settings, menus, pagination).
- **content/** — Markdown source files. Posts go in `content/posts/`. Standalone pages (like `about.md`) go in the content root.
- **static/** — Static assets copied verbatim to build output (e.g., `static/images/tvvcc.gif`).
- **themes/1995/** — Custom HTML 3.2 theme. No CSS files, no JavaScript. Uses `<FONT>`, `<CENTER>`, `<HR>` and BODY attributes for styling.
- **archetypes/default.md** — Template used when creating new content with `hugo new`.
- **public/** — Build output (gitignored). Regenerated by Hugo on build.
- **.github/workflows/deploy.yml** — CI/CD pipeline. Pushes to `main` trigger Hugo build, S3 sync, and CloudFront invalidation.

## Theme Configuration

The site uses the custom `1995` theme outputting HTML 3.2 with no CSS or JavaScript. Key params in `hugo.toml`:
- `subtitle` — Tagline shown below the site title
- `discordURL` — Discord invite link (shown in nav and footer)
- `welcomeText` — Homepage welcome paragraph

## Content Frontmatter

Posts use TOML frontmatter (`+++`) with fields: `title`, `date`, `draft`, `tags`, `categories`. The archetype template sets `draft = true` by default.
