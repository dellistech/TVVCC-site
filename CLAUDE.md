## Project Overview

Static website for the **Treasure Valley Vintage Computing Club (TVVCC)** built with Hugo and deployed on Netlify. The site uses a custom `vintage` theme that outputs HTML 3.2 for compatibility with vintage browsers (Mosaic, Netscape Navigator, Lynx).

## Commands

### Local Development
```bash
hugo server
```

### Production Build
```bash
hugo --gc --minify --baseURL "https://tvvcc.org/"
```

### Create New Post
```bash
hugo new posts/my-post-title.md
```

## Architecture

- **hugo.toml** — Primary site configuration (base URL, theme settings, menus, pagination). A legacy `config.toml` also exists but `hugo.toml` takes precedence.
- **content/** — Markdown source files. Posts go in `content/posts/`. Standalone pages (like `about.md`) go in the content root.
- **themes/vintage/** — Custom HTML 3.2 theme. No CSS files, no JavaScript. Uses `<FONT>`, `<CENTER>`, `<HR>` and BODY attributes for styling.
- **archetypes/default.md** — Template used when creating new content with `hugo new`.
- **public/** — Build output (gitignored). Regenerated by Hugo on build and by Netlify on deploy.
- **netlify.toml** — Build configuration and environment versions (Hugo 0.152.2, Dart Sass 1.93.2, Go 1.25.3, Node 22.20.0).

## Theme Configuration

The site uses the custom `vintage` theme outputting HTML 3.2 with no CSS or JavaScript. Key params in `hugo.toml`:
- `subtitle` — Tagline shown below the site title
- `discordURL` — Discord invite link (shown in nav and footer)
- `welcomeText` — Homepage welcome paragraph

## Content Frontmatter

Posts use TOML frontmatter (`+++`) with fields: `title`, `date`, `draft`, `tags`, `categories`. The archetype template sets `draft = true` by default.
